대표 카테고리별로 일부 설화를 골라 정규화 텍스트 원문을 직접 확인했습니다.

검토한 대표 항목:
- `무속신화·신격 전승`: `C_M_001`, `C_M_003`, `C_M_018`
- `마을 공동체 전승`: `C_M_002`, `C_M_005`, `C_M_011`
- `해양·어촌 전승`: `C_M_010`, `C_M_024`, `C_M_026`

## 2. 확인된 문제

### 문제 1. `가족·인간사 서사`가 너무 넓게 잡혔습니다

원인:
- 기존 규칙에서 `아들`, `딸`, `아기`, `부모` 같은 일반 단어가 들어가 있었습니다.
- 무속신화 원문에는 신의 자식, 부부, 부모 관계가 자주 나오기 때문에 많은 신화가 불필요하게 가족담으로도 잡혔습니다.

대표 사례:
- `각시당본풀이`
- `천지왕본`

판단:
- 이 축은 여전히 중요하지만, 독립 1차 카테고리라기보다 **다른 서사에 덧붙는 보조 의미 축**에 가깝습니다.

### 문제 2. `지명·지형 유래`가 장소명만으로 과하게 잡혔습니다

원인:
- 기존 규칙에 `한라산`, `산신` 같은 단어가 포함되어 있었습니다.
- GPS 주 장소가 한라산인 경우 자동 가산점도 있었습니다.

대표 사례:
- `금악본향 오일당`
- `서귀본향당본푸리`

판단:
- 단순히 산이나 오름이 배경이라는 이유만으로 `지명·지형 유래`로 분류하면 안 됩니다.
- `유래`, `설촌`, `경계`, `형성`, `지명`처럼 실제 기원 설명이 드러나는 경우에만 더 강하게 잡는 편이 맞습니다.

### 문제 3. `무속신화·신격 전승`과 `마을 공동체 전승`은 자주 함께 나타납니다

대표 사례:
- `고내본향`
- `고성신양수산본향당본풀이`
- `고전적본`

판단:
- 이 둘은 완전히 분리된 카테고리라기보다, 제주 무속신화의 상당수가 마을 공동체 전승과 결합되어 있다고 보는 편이 자연스럽습니다.
- 따라서 질문 설계 시에도 `신성한 이야기`와 `마을 전승`을 완전히 다른 층위로 분리하기보다 함께 고려할 필요가 있습니다.

## 4. 보정 후 해석

보정 후에는 다음 해석이 더 적절하다고 판단했습니다.

### 1차 주축으로 보기 좋은 것
- `무속신화·신격 전승`
- `생활민담·교훈담`
- `마을 공동체 전승`
- `해양·어촌 전승`
- `초자연 존재담`

### 공간성과 연결해 별도로 볼 것
- `지명·지형 유래`

### 주축이라기보다 보조축으로 볼 것
- `가족·인간사 서사`

즉 `가족·인간사 서사`는 카테고리 하나라기보다,  
여러 설화에 덧붙는 감정/인물 관계 태그로 다루는 편이 더 안정적입니다.

## 5. 질문 재설계에 주는 시사점

이 보정 결과를 반영하면, 질문은 다음처럼 생각하는 편이 더 적절합니다.

### 주요 구분 축
- 신성한 무속신화 계열인가
- 생활민담/교훈담 계열인가
- 초자연 존재담 계열인가
- 해양·어촌 전승인가
- 마을 공동체 전승인가
- 지명·지형 유래가 강한가

### 보조 구분 축
- 가족/인연/원한 같은 인간사 감정이 강한가

즉 `가족·인간사`는 대표 카테고리를 고르는 질문이라기보다,  
다른 서사 카테고리에 덧붙는 정서 보정 질문으로 처리하는 편이 더 자연스럽습니다.

## 6. 현재 단계 결론

표본 검토를 통해 다음이 확인되었습니다.

- 자동 태깅 v1은 큰 방향을 잡는 데는 유용합니다.
- 하지만 일부 카테고리는 규칙이 넓어 과잉 분류가 발생합니다.
- 특히 `가족·인간사 서사`는 1차 주카테고리보다 보조 태그로 해석하는 편이 더 적절합니다.
- 따라서 다음 단계에서는 `주카테고리 + 보조 정서/관계 태그` 구조로 질문과 추천 로직을 정리하는 것이 바람직합니다.
