# 실험 003. GPS 설화 1차 파일럿 태깅 메모

**날짜**: 2026-04-08  
**목적**: `002-gps-folklore-tagging-framework.md`의 태깅 기준이 실제 데이터에 적용 가능한지 확인하기 위해, 일부 GPS 설화를 먼저 시범 태깅합니다.

## 1. 파일럿 대상

초기에는 다음 기준으로 10개를 골랐습니다.

- `legend`와 `folktale`가 모두 섞이도록 선택
- 신화/초자연/해양/가족담/생활민담 성격이 고르게 보이도록 선택
- GPS가 있어 실제 코스 추천에 활용 가능한 항목만 포함

## 3. 파일럿에서 보인 초기 패턴

### 패턴 1. 무속신화와 마을 전승이 강하게 겹칩니다
- `본풀이`
- `본향당`
- `조상신`
- `당/사당`

이 계열은 단순히 "신화"로 묶기보다, `무속신화·신격 전승`과 `마을 공동체 전승`의 교집합으로 보는 것이 자연스럽습니다.

### 패턴 2. 해양 계열은 독립 카테고리 후보가 됩니다
- 바닷가 당
- 용왕/해신
- 난파선
- 도깨빗불

즉 기존의 `바다해녀`보다 조금 더 넓은 `해양·어촌 전승`이 내부 카테고리로 더 적합할 수 있습니다.

### 패턴 3. 기존 `사랑과이별`은 범위가 좁습니다
- 가족
- 계모
- 부부
- 조상
- 원한

실제 설화는 단순 연애담보다 `가족·인연`, `인간사 비극`, `권선징악` 쪽이 더 넓어 보입니다.

### 패턴 4. `오름자연`보다 `지명·지형 유래`가 더 데이터 친화적일 가능성이 있습니다
- 실제 GPS 설화에서는 `한라산`
- `바위`
- `당`
- `마을`

같이 구체적 장소성과 전승 기능이 더 중요하게 나타납니다.
