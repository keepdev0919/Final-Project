## 2. 1차 자동 태깅 결과

### 핵심 내용 태그 분포

| core topic | 건수 |
|------|---:|
| 무속신화·신격 전승 | 154 |
| 가족·인간사 서사 | 127 |
| 생활민담·교훈담 | 94 |
| 마을 공동체 전승 | 72 |
| 초자연 존재담 | 56 |
| 해양·어촌 전승 | 36 |
| 지명·지형 유래 | 35 |

해석:
- `무속신화·신격 전승`은 GPS 사용 가능 설화 풀에서 가장 큰 축입니다.
- `가족·인간사 서사`는 독립 장르라기보다 여러 설화에 넓게 겹치는 보조 축으로 보입니다.
- `생활민담·교훈담`은 민담 계열을 꽤 안정적으로 잡아냅니다.
- `해양·어촌 전승`, `지명·지형 유래`는 수는 적지만 제주 여행 코스화 관점에서 매우 유의미한 축입니다.

### 대표 카테고리 분포

`primary_category`는 넓은 축이 너무 강하게 잡히기 때문에,  
사용자 질문/UX 설계를 위해 더 특화된 `representative_category`를 별도로 계산했습니다.

| representative category | 건수 |
|------|---:|
| 무속신화·신격 전승 | 76 |
| 생활민담·교훈담 | 72 |
| 마을 공동체 전승 | 26 |
| 지명·지형 유래 | 25 |
| 해양·어촌 전승 | 11 |
| 가족·인간사 서사 | 9 |
| 초자연 존재담 | 9 |

해석:
- 실제 사용 가능한 설화는 크게 보면 `무속신화 계열`과 `생활민담 계열`의 두 덩어리가 큽니다.
- `마을 공동체`, `지명·지형`, `해양·어촌`은 수는 상대적으로 적지만 지역성/제주성 측면에서 중요합니다.
- `가족·인간사 서사`와 `초자연 존재담`은 독립 대표 카테고리로는 적지만, 질문 설계에서는 여전히 의미가 큽니다.

## 3. 자주 겹치는 태그 조합

1차 자동 태깅 결과에서 자주 함께 등장한 조합은 다음과 같습니다.

| 태그 조합 | 건수 |
|------|---:|
| 가족·인간사 서사 + 생활민담·교훈담 | 88 |
| 마을 공동체 전승 + 무속신화·신격 전승 | 72 |
| 가족·인간사 서사 + 무속신화·신격 전승 | 53 |
| 가족·인간사 서사 + 초자연 존재담 | 52 |
| 생활민담·교훈담 + 초자연 존재담 | 48 |
| 무속신화·신격 전승 + 지명·지형 유래 | 31 |
| 무속신화·신격 전승 + 해양·어촌 전승 | 26 |

이 결과는 다음을 시사합니다.

- 제주 설화는 단일 카테고리보다는 복합 성격을 가지는 경우가 많습니다.
- `무속신화`와 `마을 공동체`는 서로 자주 겹칩니다.
- `생활민담`과 `가족·인간사`도 매우 자주 겹칩니다.
- 따라서 최종 구조는 단일 선택형보다 **상위 카테고리 + 보조 태그** 구조가 더 적합합니다.

## 4. 내부 카테고리 v1 제안

현재 자동 태깅 결과를 기준으로 내부 카테고리 v1은 다음 7개를 제안합니다.

1. `무속신화·신격 전승`
2. `생활민담·교훈담`
3. `마을 공동체 전승`
4. `지명·지형 유래`
5. `해양·어촌 전승`
6. `초자연 존재담`
7. `가족·인간사 서사`

### 해석 원칙

- `무속신화·신격 전승`은 제주 설화의 가장 큰 뼈대입니다.
- `생활민담·교훈담`은 legend와 분리되는 민담 축으로 유지할 가치가 큽니다.
- `마을 공동체 전승`, `지명·지형 유래`, `해양·어촌 전승`은 제주 여행 서비스의 장소성과 직접 연결됩니다.
- `초자연 존재담`은 독립 주카테고리로 볼 수 있습니다.
- `가족·인간사 서사`는 주카테고리라기보다 여러 서사에 덧붙는 보조 정서/관계 축으로 해석하는 것이 더 안정적입니다.

즉 최종적으로는,

```text
장소성 중심 카테고리
- 마을 공동체 전승
- 지명·지형 유래
- 해양·어촌 전승

서사성 중심 카테고리
- 무속신화·신격 전승
- 초자연 존재담
- 생활민담·교훈담

보조 정서/관계 태그
- 가족·인간사 서사
```

처럼 2층 구조로 보는 것이 더 자연스럽습니다.

## 5. 기존 5개 구조와 비교

### 기존 구조
- 신화
- 도깨비
- 사랑과이별
- 바다해녀
- 오름자연

### 문제
- 사용자에게 설명은 쉽지만 실제 설화 구조와 정확히 맞아떨어지지 않습니다.
- `바다해녀`, `오름자연`, `사랑과이별`은 실제 데이터보다 이름이 더 기획 중심적입니다.
- 일부 입력 조합은 결과가 지나치게 단조롭게 매핑됩니다.

### v1 구조의 장점
- 설화 내용에서 반복되는 축이 더 직접적으로 반영됩니다.
- 장소성과 서사성을 분리해서 생각할 수 있습니다.
- 이후 질문 설계가 "예쁜 선택지"가 아니라 "실제 설화 후보를 좁히는 질문"이 됩니다.

## 6. 질문 재설계에 대한 시사점

현재 질문 구조는 `분위기 / 장소 / 일수 / 이동수단`입니다.  
하지만 자동 태깅 결과를 보면, 질문은 최소한 아래 두 축을 분리해서 설계하는 편이 더 적절합니다.

### 1) 어떤 서사 결이 좋은가
- 신성하고 기원적인 이야기
- 으스스하고 초자연적인 이야기
- 사람 사이의 관계와 감정이 강한 이야기
- 생활감 있고 재치 있는 민담

### 2) 어떤 공간 배경이 좋은가
- 바다와 해안 마을
- 마을과 당
- 산·오름·지형
- 크게 상관없음

즉 질문 설계는 단순히 "기분이 어떤가"만이 아니라,  
**서사 축**과 **공간 축**을 함께 묻는 구조로 가야 데이터와 더 잘 맞습니다.

## 7. 현재 단계의 결론

1차 자동 태깅 결과를 기준으로 볼 때,

- 현재의 고정 5테마 구조는 초기 기획안으로는 의미가 있었지만 데이터 유도형 구조라고 보기는 어렵습니다.
- GPS 사용 가능 설화 228개에서는 `무속신화`, `생활민담`, `마을 공동체`, `지명·지형`, `해양`, `초자연` 축이 주카테고리로 더 자연스럽게 드러납니다.
- `가족·인간사`는 주카테고리라기보다 보조 정서/관계 태그로 활용하는 편이 더 적절합니다.
- 따라서 다음 단계에서는 위 주카테고리와 보조 태그 구조를 기준으로 질문을 재설계하고, 이후 에이전트의 후보 선택 구조를 다시 정의하는 것이 더 타당합니다.
