코스의 매핑 상황 (실데이터)
📍 애월해안도로 ↔ 금덕본향 · 비매닛할망당 · 상가본향 · 알당
📍 사려니숲 에코힐링 ↔ 도체비 심엉 · 꿩빙에기 · 주막동산 도체빗불 · 한흘과 논흘
이 코스 매핑 8건 중 카테고리 비율 생활민담·교훈담 37.5%
무속신화·신격   25%
마을 공동체     12.5%
해양·어촌       12.5%
초자연 존재담   12.5%
사용자 입력 + 점수 계산
사용자 입력 1순위 생활민담·교훈담 → 3점
2순위 무속신화·신격   → 1점
나머지 3개          → 0점
↓점유율 × 점수 합산
계산 (3 × 37.5%) = 1.125
(1 × 25%)    = 0.25
(0 × 나머지) = 0
합계 = 1.375 × 10
↓
코스 점수 = 13.75점 score > 0 → 후보 풀에 포함 → 무작위 3개에 뽑힐 수 있음
→1,090개 큐레이션 코스 각각이 이 방식으로 점수 부여
