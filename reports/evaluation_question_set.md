# 제주 설화·민담 챗봇 평가 질문셋

## 목적

이 문서는 제주 설화·민담 RAG 챗봇의 검색 및 답변 품질을 반복적으로 점검하기 위한 고정 질문셋이다. 동일한 질문셋을 사용하면 청킹, 정규화, 프롬프트, 검색 파라미터 변경 전후의 성능을 비교할 수 있다.

## 사용 방법

1. 아래 질문을 동일한 실행 조건에서 반복 입력한다.
2. 각 질문에 대해 검색 결과, 최종 답변, 출처 형식을 기록한다.
3. 아래 판정 기준으로 `통과`, `부분 통과`, `실패`를 기록한다.

## 기본 판정 기준

- `통과`
  - 관련 질문의 경우 적절한 설화/민담을 검색하고 설명형 답변을 생성함
  - 무관 질문의 경우 자료 없음으로 응답함
  - 출처가 `제목 (코드: 번호)` 형식으로 표시됨
- `부분 통과`
  - 대체로 관련 답변이지만 검색 결과 일부가 부정확하거나 설명이 불완전함
- `실패`
  - 전혀 관련 없는 자료를 근거로 답변함
  - 무관 질문에 대해 내용을 지어냄
  - 출처가 없거나 형식이 무너짐

## 평가 질문 목록

| ID | 유형 | 질문 | 기대 동작 |
|---|---|---|---|
| Q01 | 정상/설화 | 설문대할망은 어떤 존재야? | 설문대할망 관련 설명과 출처 제공 |
| Q02 | 정상/설화 | 설문대할망 이야기에 대해 알려줘 | 설문대할망 관련 전승 요약 |
| Q03 | 정상/민담 | 닭이 나오는 제주 민담 알려줘 | 닭 관련 민담 검색 및 설명 |
| Q04 | 정상/민담 | 개와 관련된 제주 민담이 있어? | 개 관련 민담 검색 및 설명 |
| Q05 | 정상/주제 | 뱀과 관련된 제주 설화나 민담이 있어? | 뱀 관련 자료 검색 및 설명 |
| Q06 | 정상/주제 | 아들을 낳게 해준다는 이야기 있어? | 설문대할망 계열 자료 검색 기대 |
| Q07 | 정상/인물 | 한라산 산신으로 여겨지는 존재가 누구야? | 설문대할망 관련 문서 검색 기대 |
| Q08 | 정상/장소 | 한라산과 관련된 제주 설화 알려줘 | 한라산 관련 설화 설명 |
| Q09 | 정상/요약 | 제주 설화 중에서 거대한 여신 이야기를 설명해줘 | 설문대할망 중심 설명 기대 |
| Q10 | 정상/비교 | 설문대할망 관련 이야기들은 어떤 공통점이 있어? | 복수 문맥을 묶어 요약하는 답변 기대 |
| Q11 | 민담/줄거리 | 개와 닭의 원한 이야기 줄거리 알려줘 | 해당 민담 검색 및 줄거리 설명 |
| Q12 | 민담/주제 | 소가 나오는 제주 민담이 있나? | 소 관련 민담 검색 기대 |
| Q13 | 설화/주제 | 본풀이 형식의 제주 이야기 예시를 알려줘 | 본풀이 계열 자료 검색 기대 |
| Q14 | 무관 | 제주 설화에 나오는 우주선 전쟁에 대해 알려줘 | 자료 없음으로 답변 거부 |
| Q15 | 무관 | 제주 민담 속 스마트폰 이야기 알려줘 | 자료 없음으로 답변 거부 |
| Q16 | 무관 | 제주 설화에 등장하는 대통령이 누구야? | 자료 없음으로 답변 거부 |
| Q17 | 오타 | 제 주도 설화에서 가장 재밌는 설화 얘기해줘 | 오타 때문에 실패 가능, 개선 필요 여부 관찰 |
| Q18 | 오타 | 설문대할망은 어떤 존제야? | 오타 포함 질문에 대한 복원력 확인 |
| Q19 | 모호 | 가장 유명한 제주 이야기 하나만 설명해줘 | 대표 설화 하나를 무난히 추천하는지 확인 |
| Q20 | 모호 | 제주도 옛날이야기 알려줘 | 너무 넓은 질문에 대한 응답 품질 확인 |

## 기록 템플릿

아래 형식으로 질문별 결과를 누적 기록한다.

```text
ID: Q01
질문: 설문대할망은 어떤 존재야?
검색 결과: 관련 문서 5건 / 무관 문서 0건
답변 평가: 통과
출처 형식: 정상
메모: 설문대할망 관련 설명이 잘 정리됨
```

## 권장 비교 시점

- 1차 청킹 적용 후
- 정규화 규칙 변경 후
- 검색 파라미터(`k`, `max-distance`) 변경 후
- 프롬프트 수정 후
- reranker 또는 후처리 추가 후

## 추가 확장 항목

- 설화 전용 질문 10개 추가
- 민담 전용 질문 10개 추가
- 오타/띄어쓰기 변형 질문 10개 추가
- 질문별 기대 출처 코드 지정
- 정답 키워드 기반 반자동 평가 스크립트 연결
